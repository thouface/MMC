# MMC — 系统需求（技术实现规范）

## 1. 技术选型

| 层级 | 方案 | 说明 |
|------|------|------|
| UI 框架 | Flutter 3.x | 跨平台一套代码 |
| 状态管理 | Riverpod 2.x | 轻量、可测试 |
| 路由 | go_router | 声明式、深链接友好 |
| 本地存储 | Hive + SharedPreferences | 结构化 + 轻量配置 |
| 网络 | Dart:io TCP / UDP | 自建 P2P 通道 |
| 底层能力 | Platform Channel | Android NSD / iOS Bonjour、MediaProjection、权限 |

## 2. 架构分层

```
┌──────────────────────────────────┐
```

## 3. 协议设计

### 3.1 设备发现

- 自定义服务类型：`_thefool._tcp.local`
- 回退：UDP 广播（端口 54321，定时 3 秒）
- 字段：`id, name, type, os, port, file_port`

### 3.2 控制通道（TCP）

消息头：4 字节长度 + JSON 体。

```json
{
  "msgId": "uuid",
  "type": "control|file|heartbeat|info|screen|ack",
  "payload": { ... },
  "timestamp": 1234567890
}
```

### 3.3 控制指令示例

- 触控：`{ "action": "touch_down", "x": 520, "y": 800, "screenWidth": 1080, "screenHeight": 1920 }`
- 按键：`{ "action": "key", "keyCode": "BACK" }`

### 3.4 文件传输

- 分块传输：默认 1 MB 分块
- 发送流程：请求 → 确认 → 分块 → 完成
- 最大单文件：2 GB

## 4. 数据模型（Dart）

```dart
class Device {
  final String deviceId;
  final String name;
  final DeviceType type;   // phone | tablet | pc | other
  final String os;
  final String? ip;
  final int? port;
  final int? filePort;
  final DeviceStatus status; // online | offline | connecting
  final DateTime pairedAt;
}

class TransferTask {
  final String taskId;
  final String fileName;
  final int fileSize;
  final TransferDirection direction;
  final TransferStatus status;
  final int transferredBytes;
}
```

## 5. 内部 Service 抽象

```dart
abstract class DiscoveryService {
  Future<void> startDiscovery();
  Future<void> stopDiscovery();
  Stream<List<Device>> get discoveredDevices;
}

abstract class ConnectionService {
  Future<Connection> connect(Device device);
  Future<void> disconnect(String deviceId);
}

abstract class ControlService {
  Future<void> startControl(Device target);
  Future<void> sendTouch(int x, int y, String action);
  Stream<ScreenFrame> get screenStream;
}

abstract class FileTransferService {
  Future<TransferTask> sendFile(Device target, String filePath);
  Future<void> acceptTransfer(String taskId);
  Stream<TransferTask> get transferTasks;
}
```
